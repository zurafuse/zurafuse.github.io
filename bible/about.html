THE SCRIPTURES

This project is a Bible translation that I am working on (it will probably be years before it is complete, if ever). Visit the Github page at https://github.com/zurafuse/BibleApp.

When you hover over a verse or touch it, the background should be gray or blue. If it is shaded blue, that is a verse that I have already translated. If the hue is gray, it is probably the King James Version and I have not gotten to it yet.

The site was created using jQuery. Each section of the Bible is a JSON object that is parsed according to your selection.
